# ZCanFeedbackBiDi
Feedback decoder for BiDi with ZCan interface to Z21 based on Esp32

This project is based on the work of:
- Philipp Gathow (https://pgahtow.de/w/R%C3%BCckmeldung 8xRailCom-Detektor)
